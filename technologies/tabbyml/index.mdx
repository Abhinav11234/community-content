# TabbyML

TabbyML is a company that specializes in building an AI-powered code assistant to help developers write code faster and more efficiently. Their product, Tabby, draws context from comments and code to suggest lines and whole functions instantly, and is available as an extension for Visual Studio Code and VIM. TabbyML's approach to AI development is centered around openness and transparency.

Visit the [TabbyML website](https://www.tabbyml.com/)
